== Installation ==

To install WP Landing Page plugin to do :

1. Click here to download plugin
2. Go to your WP site’s plugin page on your Dashboard and select “add plugin” on the top left. Then click on “Upload” again on the top left
3. Select the file and press “Install”. Installation may take up to 30-45 seconds.
4. Finally, activate the plugin and you are ready to go!
